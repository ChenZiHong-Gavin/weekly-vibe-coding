# 光影皮影戏 (Shadow Puppet)

手势操控数字皮影戏。通过摄像头识别手掌与手指动作，实时操控皮影人物的身体移动、头部转动和手臂挥舞。捏合、挥手、握拳等手势可触发金光、花瓣、烟雾等舞台特效。

## 功能

- **手势追踪** — MediaPipe Hands 识别 21 个手部关键点，实时映射到皮影关节
- **角色选择** — 武生、花旦、孙悟空、神龙、老生五种皮影，各有不同的物理个性
- **场景切换** — 多种传统舞台场景
- **手势特效** — 捏合触发金光、挥手触发花瓣、握拳触发烟雾
- **音效系统** — 锣、鼓、木鱼等传统乐器音效
- **多种模式** — 自由表演模式随时互动，影子模式呈现剪影效果

## 手势操作

| 手势 | 控制 | 说明 |
|------|------|------|
| 手掌平移 | 身体移动 | 掌心在画面中移动，皮影跟随走动 |
| 食指摆动 | 头部转动 | 食指左右偏移掌心，控制头部方向 |
| 拇指伸展 | 左臂挥舞 | 拇指向外侧伸展，挥动左臂 |
| 小指伸展 | 右臂挥舞 | 小指向外侧伸展，挥动右臂 |
| 捏合 | 金光特效 | 拇指食指尖端靠近 |
| 挥手 | 花瓣特效 | 手掌左右快速摆动 |
| 握拳 | 烟雾特效 | 五指握拳保持 0.6 秒 |

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- Zustand 状态管理
- MediaPipe Hands 手势识别
- Canvas 2D 渲染（OffscreenCanvas 缓存、Path2D 预计算）
- Web Audio API 音效合成

## 安装与运行

```bash
cd apps/shadow-puppet
npm install
npm run dev
```

打开浏览器访问 `http://localhost:8080`，允许摄像头权限后即可开始操控。

## 构建

```bash
npm run build
```

生产构建输出至 `dist/` 目录。
