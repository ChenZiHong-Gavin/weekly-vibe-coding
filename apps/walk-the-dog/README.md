# 今天谁遛狗 (Walk The Dog)

限时遛狗挑战游戏 🐕 — 选择你的狗狗、场景和道具，在 180 秒内完成一次完美的遛狗之旅。小心狗狗暴走！

## 功能

- **5 种狗狗** — 柯基、哈士奇、柴犬、金毛、法斗，各有不同的服从、力量、体力、好奇心属性
- **4 个场景** — 小区公园、商业街、湖边步道、狗狗公园，各有不同的随机事件
- **3 种道具** — 飞盘（连击神器）、伸缩绳（更自由但难控）、训练绳（口令成功率提升）
- **实时遛狗模拟** — WASD / 方向键 / 触屏拖动操控主人，牵绳物理 + 狗狗自主行为 AI
- **多维状态** — 快乐、运动、文明、清洁、兴奋，兴奋满格狗狗会暴走！
- **随机事件** — 小动物窜出、玩具诱惑、人流拥挤、下雨积水、车辆冲撞、其他狗狗社交
- **连击积分** — 连续成功互动积累连击倍数，冲刺高分
- **6 种口令** — 换绳长、等等、回来、不许吃、零食、捡/互动

## 操作说明

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | WASD / 方向键 / 触屏拖动 | 操控主人移动 |
| 换绳长 | Shift | 短 / 中 / 长 三档切换 |
| 等等 | 空格 | 让狗狗停下来 |
| 回来 | R | 召回狗狗 |
| 不许吃 | F | 阻止狗狗吃地上的东西 |
| 零食 | Q | 投喂零食安抚情绪（限 3 次） |
| 捡/互动 | E | 铲屎或与场景互动 |
| 扔飞盘 | 空格²（选飞盘时）| 扔出飞盘让狗狗叼回 |

## 技术栈

- 纯原生 HTML/CSS/JS，单文件，零依赖
- Canvas 2D 渲染（双倍分辨率）
- 牵引绳物理模拟（张力、缠树）
- 触屏虚拟摇杆支持
- 狗狗自主行为 AI（刺激评估 + 状态机）

## 运行

直接用浏览器打开 `index.html`，或部署到任意静态服务器。

```bash
# 本地开发
cd apps/walk-the-dog
npx serve .
```

## 构建

无需构建，单文件即用。
