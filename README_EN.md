## About the Project

`weekly-vibe-coding` is an experimental project: one Vibe Coding session per week, using prompts to rapidly build and realize fun ideas.

Each app is an independent sub-project under the `apps` directory.

Follow my WeChat official account: 【无所事事的执念】for coding insights.

## How to Run

Example: running the `dice-verse` app locally.

1.  Clone the repo:
    ```bash
    git clone https://github.com/chenzihong-gavin/weekly-vibe-coding.git
    cd weekly-vibe-coding
    ```

2.  Enter the app directory and install dependencies:
    ```bash
    cd apps/dice-verse
    npm install
    ```

3.  Start the dev server:
    ```bash
    npm run dev
    ```
