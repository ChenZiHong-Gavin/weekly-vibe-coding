# Prompt for Spellbook

```
做一个哈利波特魔法咒语练习应用。

核心玩法：用户对着麦克风念出咒语，系统通过音素识别判断发音是否正确，正确则播放对应的魔法动画效果。

功能需求：
1. 前端以卡片形式展示咒语列表，按书籍分类筛选
2. 点击咒语卡片进入练习模式，显示咒语名称、发音提示和说明
3. 录音时展示实时波形可视化
4. 录音结束后上传到后端，返回发音匹配分数
5. 匹配成功则播放该咒语专属的魔法动画效果（如荧光闪烁、除你武器等）

技术栈：
- 前端：React + Vite + TypeScript + shadcn/ui + Framer Motion
- 后端：FastAPI + allosaurus（音素识别）+ Levenshtein（相似度计算）
- 音频处理：AudioWorklet 采集 PCM，封装为 WAV 上传
```
