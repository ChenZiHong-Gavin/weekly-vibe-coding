# Prompt for Paper2Novel

```
做一个将 arXiv 学术论文转化为小说的网页工具。

功能需求：
1. 用户输入 arXiv 论文链接，自动抓取论文内容（通过 ar5iv HTML 版本）
2. 提供 10 种小说风格选择：科幻、奇幻、悬疑推理、武侠、言情、恐怖、幽默讽刺、童话、仙侠修真、战争史诗
3. 将论文按段落分块，逐章调用 LLM 改写为对应风格的小说
4. 支持 SSE 流式输出，实时渲染生成的小说章节
5. 保留论文的所有核心知识点，但用角色、对话、场景来承载
6. 纯前端实现，用户自行配置 OpenAI API Key（存 localStorage）
7. 支持自定义 API Base URL 和模型名称
```
