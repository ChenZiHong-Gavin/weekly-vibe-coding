# Shadow Puppet - 手势操控数字皮影戏

## 项目描述

做一个交互项目，用摄像头手势识别操控数字皮影戏。MediaPipe Hands 识别手掌和手指动作，映射到皮影人物的身体移动、头部转动和手臂挥舞。不同手势触发舞台特效。

## 技术要求

- 使用 MediaPipe Hands 进行实时手势识别，追踪 21 个手部关键点
- 使用 React 18 + TypeScript + Vite 构建
- Canvas 2D 渲染皮影人物，支持 OffscreenCanvas 缓存和 Path2D 预计算
- 五指映射：掌心→身体移动，食指→头部转动，拇指→左臂挥舞，小指→右臂挥舞
- 手势识别：捏合→金光特效，挥手→花瓣特效，握拳→烟雾特效
- 多种皮影角色：武生、花旦、孙悟空、神龙、老生，各有不同物理个性
- 使用 Web Audio API 合成锣、鼓、木鱼等传统乐器音效
- 当摄像头不可用时自动切换为鼠标控制模式
- 支持影子模式（剪影效果）
