# Prompt for Louis XVI Sort

```
做一个路易十六排序算法的可视化网页。

背景：Louis XVI Sort 是一个抽象/恶搞排序算法——从右到左扫描数组，如果左边的元素比右边大，就把左边的"砍头"（值降为右边的值）。名字来源于法国大革命中被送上断头台的路易十六。

功能需求：
1. 柱状图可视化数组，每一步高亮当前比较的元素
2. 被"砍头"的元素有红色闪烁 + 断头台斩落动画效果
3. 左侧显示伪代码面板，实时高亮当前执行的行
4. 底部显示当前步骤的文字说明（中文）
5. 支持播放/暂停、单步执行、重新生成数组
6. 支持调节速度（慢/正常/快）和数组大小
7. 使用 Web Audio API 合成音效：比较时的轻微 tick、砍头时的金属斩击+落地声、排序完成的胜利号角
8. 技术栈：React + TypeScript + Vite + Tailwind CSS + Framer Motion
```
