# Prompt for Pretext Solar System

```
使用 @chenglou/pretext 排版引擎，做一个太阳系主题的动态文字排版演示页面。

功能需求：
1. 深色太空背景，用 Canvas 绘制闪烁星空
2. 中央放置太阳，八大行星沿椭圆轨道围绕太阳运行，每个行星有光晕效果和名称标签
3. 用 SVG 绘制轨道线
4. 一篇关于太阳系的英文长文，使用 pretext 引擎排版为报纸式双栏布局
5. 核心特效：文字实时环绕行星——当行星运动到文字区域时，文字自动避让行星，形成流动的排版效果
6. 排版细节：大标题、首字下沉（Drop Cap，金色大写字母）
7. 支持速度调节滑块，点击页面暂停/继续
8. 显示性能统计（帧数、每帧排版行数、耗时）
9. 技术栈：TypeScript + Vite + @chenglou/pretext
```
