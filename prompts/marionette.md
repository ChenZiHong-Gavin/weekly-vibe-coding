# Marionette - 手势操控提线木偶

## 项目描述

做一个交互项目，可以用手势操作提线木偶。MediaPipe 识别手指头，对应木偶线上的控制关节。

## 技术要求

- 使用 MediaPipe Hands 进行实时手势识别，追踪五个指尖位置
- 使用 Verlet 积分物理引擎模拟提线木偶的骨架和约束
- HTML5 Canvas 2D 渲染，手绘布偶风格（红色布身体、黄色辫子、纽扣眼、绿色缝线、木质十字架）
- 手指到木偶的映射采用掌心相对定位：掌心位置决定木偶整体位置，各手指相对掌心的偏移控制对应关节
- 手指映射：拇指→左脚，食指→左手，中指→头，无名指→右手，小指→右脚
- 当摄像头不可用时自动切换为鼠标控制模式
- 纯前端实现，单 HTML 文件，无需构建工具
