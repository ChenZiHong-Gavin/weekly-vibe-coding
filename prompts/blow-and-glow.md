# Prompt for Blow & Glow

```
做一个"吹烛焚香"交互网页，单文件 HTML，使用 Three.js。核心概念：

1. 选场景界面 — 4 个场景可选：佛堂供桌（3烛3香）、书房案头（1烛1香）、石庭夜灯（2烛2香）、洞穴祭坛（5烛3香）
2. Three.js 3D 场景 — 所有模型纯几何体搭建（圆柱体蜡烛、粒子火焰、粒子烟雾、几何体环境）
3. 麦克风吹气检测 — Web Audio API 实时分析麦克风音量+频谱平坦度，区分吹气（宽带噪声）和说话（谐波结构），过滤瞬态噪音
4. 火焰粒子系统 — 每根蜡烛独立 70 粒子池，AdditiveBlending 叠加渲染，白→黄→橙→红色彩梯度，上升+随机抖动+吹气Z轴偏移
5. 烟雾粒子系统 — 每根香独立 50 粒子池，NormalBlending 半透明，缓慢上升扩散，吹气加速逸散
6. 吹气三级响应 — 轻吹火焰摇曳、中吹火焰剧烈摆动+烟散、重吹蜡烛熄灭→余烬→缓慢复燃
7. 吹气力度可视化 — 左上角圆环指示器随吹气力度缩放，颜色绿→黄→红过渡
8. 鼠标拖拽旋转视角 + 滚轮缩放 + WASD 平移

视觉风格：暗色调写实风，ACES 色调映射，PCFSoftShadowMap 阴影，点光源随火焰抖动。
```
