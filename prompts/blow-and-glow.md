# Prompt for Blow & Glow

```
做一个"烟"交互网页，单文件 HTML。核心概念：

1. 全屏烟雾场 — Canvas 2D 渲染 700 个半透明软边粒子，Perlin 噪声驱动自然翻涌上升，AdditiveBlending 叠加，暗色背景+暗角
2. 摄像头嘴部追踪 — MediaPipe Face Mesh 识别人脸 468 个关键点，提取嘴唇中心坐标（landmark 13+14），镜像映射到屏幕位置
3. 麦克风吹气检测 — Web Audio API 分析 RMS 音量+频谱平坦度，区分吹气（宽带噪声）和说话（谐波结构）
4. 吹气驱散烟雾 — 嘴部在屏幕的对应位置周围 180px 范围内的烟雾粒子被径向推力推开，推力随距离高斯衰减+随机湍流
5. 嘴部指示器 — 屏幕上淡淡圆环标记嘴部位置，吹气时放亮+放大，无吹气时低调
6. 摄像头预览 — 右下角小圆形窗口显示镜像摄像头画面，红点标记嘴唇位置
7. UI 自动隐藏 — 6秒无操作后状态栏和摄像头预览淡出，鼠标移动恢复
8. 粒子物理 — 速度阻尼 0.96，屏幕边缘 wrap 环绕，多层视差（3层上升速度不同）

视觉风格：暗黑极简，烟雾灰白半透明柔和，嘴部标记微弱环形，无多余 UI。
```
